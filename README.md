# tsdt-sample

### setup

```terminal
$ npm install -g yarn typescript typings
$ git clone https://github.com/ina6ra/tsdt-sample
$ cd tsdt-sample
$ yarn run init
```

### compile

```terminal
$ yarn run tsc
#success(no error)
```

### test

```terminal
$ yarn run test
#fail
```
